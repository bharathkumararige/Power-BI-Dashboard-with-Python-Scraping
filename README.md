# 📊 Power BI Dashboard with Python Web Scraping

This project demonstrates how to scrape real-time data from the web using Python and create interactive dashboards in Power BI. It is a complete data pipeline from data extraction to visualization, perfect for data enthusiasts, analysts, and developers.

## 🧰 Technologies Used

- **Python**: `requests`, `BeautifulSoup`, `pandas`, `selenium` (optional)
- **Power BI Desktop**
- **Power Query Editor**

## 🚀 Project Overview

- Scrapes data from [].
- Cleans and formats data using pandas.
- Exports the cleaned data to CSV.
- Imports the CSV into Power BI to create visualizations and dashboards.
